# 视频压缩工具

`scripts/compress-videos.mjs` 提供批量压缩 MP4 视频至 720p 的功能，基于 FFmpeg。

## 基本用法

```bash
# 压缩指定目录下所有 .mp4 文件
npm run compress -- ./path/to/videos

# 等效写法
node scripts/compress-videos.mjs ./path/to/videos
```

## 常用选项

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-o, --output-dir <dir>` | 输出目录（不指定则在同目录生成 `*_720p.mp4`） | 输入目录 |
| `--crf <0-51>` | 质量控制，越低质量越高 | 23 |
| `--preset <name>` | 编码速度预设 | medium |
| `--height <px>` | 目标高度 | 720 |
| `--dry-run` | 仅显示命令，不执行 | — |

## 示例

```bash
# 压缩到指定输出目录
npm run compress -- -o ./output ./recordings

# 使用更高 CRF（更小体积，质量稍低）
npm run compress -- --crf 28 ./recordings

# 压缩到 480p
npm run compress -- --height 480 ./recordings

# 预览将执行的命令
npm run compress -- --dry-run ./recordings
```

## 行为说明

- 跳过文件名已包含 `_720p` 的文件，避免重复压缩
- 跳过高度已低于目标值的视频
- 输出文件命名为 `{原文件名}_{目标高度}p.mp4`
- 默认使用 `libx264` + `aac 128k` 编码

## 前置要求

系统需安装 FFmpeg，或通过 `npm run postinstall` 安装 bundled 版本（`ffmpeg-static`）。
